Ciclos

Sentencia repetitiva de código.

Los ciclos, bucles o sentencias repetitivas son fragmentos de código que se ejecutan varias veces hasta que la condición asignada a dicho ciclo sea cumplida, los dos tipos de bucles más utilizados en programación son el ciclo while y el ciclo for.

El primer ciclo, for se define usando la palabra reservada for de la siguiente manera:

for([variable_de_recorrido ; condicion_de_variable; incremento_de_variable_de_recorrido ]) { 
	[Lista de instrucciones separadas por ejecutar] 
}

De esta manera el ciclo for funciona con una estructura definida en su cabecera mediante un condicional, en este caso el condicional se define inicialmente por medio de una variable que definirá un rango máximo, y mediante un contador que, procesado mediante el condicional definirá la cantidad de ciclos que el ciclo for ejecutará.

En el siguiente ejemplo suponga que definimos e inicializamos una variable texto texto con la cadena de caracteres vacia, y queremos realizar un ciclo que en cada iteracion le vaya agregando a esta variable, la misma mas un numero nuevo cada iteracion.

var texto = "";

Ahora vamos a escribir un programa que rellene a la variable texto con los numeros del rango de 5 a 25.

var texto = ""
var i;
for (i = 5; i <= 25; i++) {
	texto += i + "<br>";

El segundo ciclo, while se define usando la palabra reservada while de la siguiente manera:

while([Condición]) { 
	[Lista de instrucciones separadas por ejecutar] 
}

De esta manera el ciclo while funciona con un unico condicional que permite ejecutar un bloque de código repetidas veces hasta que el condicional deje de satisfacerse. Es decir, si en el siguiente ciclo while(true) -> ejecuta & while(false) -> termina.

En el siguiente ejemplo vamos a definir una variable texto como cadena de caracteres vacia.

var texto = "";

Ahora vamos a escribir un programa que en cada iteracion del ciclo while va a ir agregando el caracter 'a' a la variable texto hasta que la longitud del mismo sea 50.

var texto = ""
while (texto.length < 50) {
	texto += "a";
}
document.getElementById("demo").innerHTML = texto;
